# AI-Based Vehicle Speed Detection & Overspeed Violation Management System

This project is a full-stack, AI-powered traffic surveillance system designed to automatically detect vehicles from video streams, estimate their real-time speed, and identify overspeed violations. The system aims to reduce manual traffic monitoring by leveraging computer vision, deep learning, and modern web technologies.
